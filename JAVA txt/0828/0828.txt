리스트 맵
리스트는 인덱스로
맵은 키로

맵이 키로 지정하는게 찾아볼 때라던가 관리가 쉽다.

json은 맵 형식으로 되어 있다.
API 요청하면 그 결과 값이 날라오는 것

리스트
넣기 .add("")
꺼내기 .get(인덱스번호)

맵
넣기 .put(키값,value값)
꺼내기 .get(키값)

맵에서 출력할 시에는 
2중 포문을 사용할 때도 그리고 그 때는
리스트 객체를 만들어 담아서 출력하는 방식으로


메소드

메소드 작성시 변수타입 지정했다면
변수타입에 맞는 값으로 넣어줘야함

메소드 작성시 변수 갯수 지정했다면
변수 넣을 때도 갯수 맞춰서 넣어 줘야 한다.

변수를 많이 사용하는게 읽기 쉬울 수도 있으니 고려 할 것.

리턴
Stirng 경우 null;
int 경우 0;
으로 기본값 설정

Map<String,List<String>> 이것도 타입
List<String> 이것도 타입
